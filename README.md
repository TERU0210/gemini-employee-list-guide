# gemini-employee-list-guide

従業員リストから「ラベル用リスト」と「受付用リスト」を Gemini で作るための手順書です。

## この手順書でできること

- お手元の従業員リスト（Excel）をもとに、次の 2 種類のリストを作れます。
  - **ラベル用リスト**: 店舗名と従業員（苗字のみ）を 2 列で並べた一覧
  - **受付用リスト**: （準備中）
- Gemini に CSV を直接貼り付けて変換させる方法を紹介しています。
- AI を使うのが初めての方でも、順番どおりに進めれば同じ結果が作れるように書いています。

## 事前に用意するもの

- 従業員リストの Excel ファイル（店舗名と従業員名が入っているもの）
- インターネットにつながった PC
- Gemini（<https://gemini.google.com>）が使える Google アカウント

## 1. ラベル用リストの作り方

### 手順

1. **従業員リストを用意します**
   - まずは従業員リストを Excel で作成してください（この部分は手動で作業します）。
2. **Excel ファイルを CSV に変換して保存します**
   - Excel の画面左上にある「ファイル」タブをクリックします。
   - 「エクスポート」をクリックし、「ファイルの種類の変更」を選びます。
   - 「その他のファイルの種類」から「CSV(カンマ区切り)」を選択して保存します。
   - 補足: CSV とは、データをカンマで区切っただけのシンプルな表形式のファイルのことです。
3. **CSV ファイルをメモ帳で開き、中身をすべてコピーします**
   - 保存した CSV ファイルを右クリックし、「プログラムから開く」→「メモ帳」を選びます。
   - `Ctrl + A` ですべて選択し、`Ctrl + C` でコピーします。
4. **Gemini にプロンプトとコピー内容を貼り付けて依頼します**
   - Gemini（<https://gemini.google.com>）を開きます。
   - 下の「Gemini に貼り付けるプロンプト」をコピーして貼り付け、続けて手順 3 でコピーした CSV の中身を貼り付けて送信します。
   - 出力が返ってきたら、データが全件含まれているか、誤りがないかを必ず目視で確認します。
5. **Google スプレッドシートにエクスポートします**
   - Gemini の出力結果の末尾にある「Google スプレッドシートにエクスポート」ボタンをクリックします。
6. **作成されたスプレッドシートを開きます**
   - 画面の左下に「スプレッドシートを開く」のリンクが表示されるので、これをクリックします。
7. **Excel 形式でダウンロードします**
   - スプレッドシートの「ファイル」→「ダウンロード」→「Microsoft Excel (.xlsx)」の順に選びます。
   - ダウンロードされたファイルが、ラベル用の Excel ファイルになります。

### ポイント

- CSV ファイルに変換してから、その中身を直接プロンプトに貼り付けるのが **一番精度の高いやり方** でした。
- Excel ファイルをそのまま添付したり、画面の一部だけをコピーして貼り付けたりすると、データの取りこぼしが起きやすくなります。

### Gemini に貼り付けるプロンプト

以下をそのままコピーして、Gemini に貼り付けてください。続けて CSV の中身を貼り付けて送信します。

````text
# 依頼内容
貼付したデータから、指定のフォーマットで一覧表を作成してください。

# 抽出ルール
1. 1行目のヘッダー行は無視してください。
2. 1列目のデータ（店舗名）を1列目に出力してください。
3. 2列目以降に横に並んでいる従業員名をすべて抽出し、2列目に出力してください。
4. 従業員名は「苗字 名前」の形式になっていますが、「苗字のみ」を抽出してください。

# 出力フォーマット
- 以下の2列の表形式で出力してください。
  - 1列目：店舗名
  - 2列目：従業員名（苗字のみ）
- 1店舗につき複数人の従業員がいる場合は、一人一行ずつ作成してください。
- 並び順は「1店舗目の一人目、二人目...」の順で作成してください。

# 徹底事項（重要）
- 途中で省略せず、必ず全店舗（店舗1〜店舗20など、ファイル内の末尾まで）のデータをすべて出力してください。
- 出力前に、ソースデータ内の全行が含まれているか自己検閲を行い、漏れがないことを確認してから回答してください。
- Pythonコードの提示は不要です。結果の表のみを出力してください。
````
